## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git installed on your machine
- A code editor (VS Code, Sublime Text, etc.)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ElOlamirewaju/Olanrewaju-portfolio-website.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Olanrewaju-portfolio-website
   ```

3. **Open in your browser**
   - Simply open `index.html` in your browser, or
   - Use a local server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (with http-server)
     npx http-server
     ```

4. **View in browser**
   - Navigate to `http://localhost:8000`

## ğŸ“ Customization

### Updating Content

1. **Personal Information**: Edit the text in `index.html`
2. **Styling**: Modify colors and fonts in `styles.css`
3. **Projects**: Add your work samples to the `assets/` folder
4. **Interactions**: Update animations in `script.js`

## ğŸŒ Deployment

### Deploy to GitHub Pages

1. **Enable GitHub Pages**
   - Go to your repository settings
   - Scroll to "Pages" section
   - Select `main` branch as source
   - Click "Save"

2. **Access your site**
   - Your site will be live at: `https://elolamirewaju.github.io/Olanrewaju-portfolio-website/`
   - Usually takes 1-2 minutes to deploy

### Alternative Deployment Options

- **Netlify**: Drag and drop deployment
- **Vercel**: Connect GitHub repo for automatic deploys
- **Cloudflare Pages**: Free hosting with CDN
